Yii2 boilerplate
================

# Prerequisites

1. PHP >=7
2. [composer](https://getcomposer.org) installed

# Installation
0. Clone this project
1. Install *globally* `composer-asset-plugin`:
	```bash
	$ composer global require "fxp/composer-asset-plugin"
	```
2. Then run:
	```bash
	$ composer install
	```
